Option G

Network graph (force-directed)

Each doc is a circle. Edges show replaces / amends / implements. Drag nodes, scroll to zoom. Click a node to highlight its 1-hop neighborhood. (Static mock below — live version is interactive.)
Luật NĐ TT QĐ ━━ thay thế (replace) ┄┄ implements
Strength: Most "system-thinking" view. Shows all relationships simultaneously. Easy to spot orphans (no edges) and clusters. Interactive zoom/drag in the live version.
Weakness: Hard to read at high node count (the full Điện lực set has ~50 nodes — would need filters). No natural sub-sector grouping unless we cluster by it (which adds complexity). Mobile-unfriendly.
