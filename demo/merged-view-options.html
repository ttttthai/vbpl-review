Strength: Most human-readable view. Tells the regulatory story chronologically with context. Easy for non-legal readers. Excellent for "what changed recently?" queries.
Weakness: Auto-generating clean narrative prose is hard — requires templated descriptions per event type. Doesn't show structural hierarchy at all. Scaling to 50+ events means a long scroll.
Option I

Spider / radial system

Anchor Luật at the center. Sub-sectors radiate outward as "spokes". Distance from center = age (newest closer). Tier shown by color. Replace edges drawn between docs.
Strength: Striking, distinctive visual. Sub-sectors at a glance via spokes. Newer docs are closer to anchor = intuitive. Predecessor chains visible per ray. Photo-shareable.
Weakness: Hard to add many docs per ray without crowding. Mobile-unfriendly. Reading text labels at 360° angles is fatiguing. Best for "showpiece" / executive summary rather than working tool.
Option J

Side-by-side: chronology ↔ hierarchy

Two panes. Left = chronological list (Tiến hóa). Right = sub-sector pyramid (Hệ thống). Hover or click a doc on either side, the matching doc on the other side highlights. Linked navigation.

Option K

Sticky first column = sub-sector. Subsequent columns = tier (NĐ / TT / QĐ). Each cell holds the active doc as a pill plus past docs as smaller secondary list. Sortable, filterable, scrollable. Most analytical.
